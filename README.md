# Theme Skill

Edits Shopify JSON template files using your theme's actual existing block and section types. This skill is for updating, editing, and tweaking Shopify JSON, not writing custom Liquid or creating new theme blocks/sections.

```bash
npx skills add BrickspaceLab/theme-skill
```

Once installed it's easy - just use natural language to update Shopify JSON templates.

```
Update my index.json template with...
```

```
Use the screenshot and create a new page.about.json template
```
